# AI for TadaCraft

## Development

1. Run system using docker-compose.

```bash
docker-compose pull && \
  docker-compose up -d
```

2. Setup python development environment.

```bash
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
```

3. Write your AI into main.py and run it.

```bash
python main.py
```
